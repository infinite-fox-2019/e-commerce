# Game Stop - E-commerce

## Link Deploy
http://gamestop.andreassosilo.co/

## How to use locally

### On server
```javascript
$ npm install
$ npm run dev
```

### On client
```javascript
$ npm install
$ parcel index.html
```
Access client via `http://localhost:1234`

Access server via `http://localhost:3000`
